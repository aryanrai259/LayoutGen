# Floor Plan AI Frontend

A clean, minimal React frontend for the AI Floor Plan Designer application.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **JavaScript** - Programming language
- **CSS** - Styling

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

## Backend

The backend is a separate service. Make sure your backend is running on the URL specified in `.env` (default: `http://localhost:8000`).

## Project Structure

```
src/
├── api/          # Backend API functions
├── components/   # Reusable React components
├── pages/        # Page components
└── styles/       # CSS files
```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
